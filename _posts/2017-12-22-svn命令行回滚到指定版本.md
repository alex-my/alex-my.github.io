# 1 回滚

- 根据日志查出的版本号进行回滚

  ```text
  // 放弃当前修改
  svn revert -R .
  // 更新
  svn up
  // 查看最近5条svn log日志
  svn log -l 5
  // 从r608回滚到r602
  svn merge -r 608:602 ""
  ```
